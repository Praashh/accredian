
# Accredian


Earn by Refer, Accredian is a platform where people can refer course and anything with their code and earn money up to 15k INR.


## Local setup

- Clone the repo.

```bash
   git clone https://github.com/<yourusername>/accredian.git
```

- Install dependencies

```bash
  npm install 
```

- Start the server.
```bash
  npm run dev
```
